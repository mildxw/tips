Описание проекта

Данное приложение разработано на WPF и предназначено для расчёта чаевых.

Пользователь вводит:

* сумму счёта;
* количество гостей;
* процент чаевых.

В приложении доступны следующие варианты чаевых:

* без чаевых;
* 5%;
* 10%;
* 15%.

После нажатия кнопки «Рассчитать» программа выводит результат на экран: сумму счёта, сумму чаевых, итоговую сумму и сумму на каждого гостя, если гостей больше одного.

Чаевые рассчитываются на весь счёт, а не отдельно на каждого гостя.

Структура проекта

В проекте реализованы:

* XAML-интерфейс приложения;

* отдельный класс логики logic;
* автоматизированные тесты MSTest.

Логика приложения

Основная логика расчётов вынесена в отдельный класс logic.

В классе реализованы методы:

* CalculateTip — расчёт суммы чаевых;
* CalculateTotal — расчёт итоговой суммы с чаевыми;
* CalculatePerGuest — расчёт суммы на каждого гостя;
* IsGroupBill — проверка, нужно ли делить счёт между несколькими гостями.

При некорректных входных данных методы выбрасывают исключение ArgumentException.

Обработка ошибок

В приложении реализована проверка вводимых данных.

Если пользователь вводит некорректную сумму счёта или количество гостей, приложение не завершает работу аварийно, а выводит понятное сообщение об ошибке через MessageBox.

Проверяются следующие ситуации:

* сумма счёта меньше или равна нулю;
* количество гостей меньше или равно нулю;
* введены нечисловые значения;
* выбран некорректный процент чаевых.

Автоматизированное тестирование

Для проверки работы логики был создан отдельный тестовый проект MSTest.

Тестами проверяются:

* корректные входные данные;
* расчёт чаевых 0%, 5%, 10% и 15%;
* расчёт итоговой суммы;
* расчёт суммы на одного гостя;
* расчёт суммы при нескольких гостях;
* граничные значения;
* некорректные данные и выбрасывание исключений.

В тестах используются:

* Assert.AreEqual;
* Assert.AreNotEqual;
* Assert.IsTrue;
* Assert.IsFalse;
* Assert.ThrowsException.

Вывод

В ходе выполнения работы было разработано WPF-приложение «Калькулятор чаевых».
Приложение позволяет рассчитать сумму чаевых, итоговую сумму счёта и сумму на каждого гостя.
